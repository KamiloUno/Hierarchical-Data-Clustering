# Hierarchical Data Clustering

C++ implementation of a hierarchical clustering algorithm for D-dimensional data points with a CLI interface.

## Overview

This project provides a robust and efficient implementation of hierarchical clustering algorithms designed to work with data points in arbitrary dimensions. The command-line interface makes it easy to cluster your data and analyze the results.

## Features

- **Multi-dimensional Support**: Cluster data points in D-dimensional space
- **Efficient Algorithm**: Optimized hierarchical clustering implementation
- **CLI Interface**: User-friendly command-line interface for easy data processing
- **Flexible Configuration**: Customizable parameters for different clustering scenarios
- **Performance Focused**: Written in C++ for high performance and low memory overhead
